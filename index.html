Layery.jsv0.0.01 beta

jQuery + CSS3 Transition based UI Framework.

Usage

Initialize

- add 'layery' in top-level DOM

<div class="top-level-DOM" layery>
</div>

- script

<script>
  layery.initOnLoad();
</script>

- It restructs HTML DOM

<div layery-glass>
  <div glass-bg>
    <!-- copied top-level DOM -->
    <div class="top-level-DOM" layery>
    </div>
  </div>
  <div glass-content>
    <!-- layery modal layer -->
  </div>
</div>
<div layery-stage>
  <!-- original top-level DOM -->
  <div class="top-level-DOM" layery>
  </div>
</div>

Blur

- Blur effect

<script>
  layery.toggleBlur();
</script>
